Heroku:

- Buat Akun Di Heroku
- Install Heroku-cli
- Buat file composer.json dengan syntax {}
- Buat file index.php <?php include_once("home.html"); ?>

Jalan perintah-perintah berikut di terminal:
- git init
- heroku login
- heroku create nama-app
- git add .
- git commit -m "initial commit"
- git push heroku master
